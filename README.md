# Support Ticket Classification

This project classifies customer support tickets automatically using Machine Learning.

The system analyzes customer messages and categorizes tickets based on priority and issue type.

## Features
- Ticket classification
- Text preprocessing
- Priority prediction
- NLP based analysis

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK

## Project Outcome
This project helps support teams respond to customer issues faster and more efficiently.
